Install Stack as an App

Chrome / Edge

  1. Tap the menu icon ⋮ (three dots)
  2. Select "Install app" or "Add to Home screen"
  3. Tap "Install" to confirm

Safari (iOS)

  1. Tap the Share button (square with arrow)
  2. Scroll down and tap "Add to Home Screen"
  3. Tap "Add" to confirm

Firefox

  1. Tap the menu icon ⋯ (three dots)
  2. Select "Install" or "Add to Home screen"
  3. Tap "Add" to confirm

Installing as an app provides offline access, faster loading, and a full-screen experience.

How Stack Works

Accounts track your assets and debts. Positive amounts are assets, negative are liabilities.

Expenses are items with a due date and budgeted amount. Check "Recurring" for expenses that repeat each month on the same day.

Available = Assets minus Liabilities minus all Expenses.

Auto Fill suggests spending allocations based on upcoming due dates and past patterns.

Gist Sync Data is stored locally by default. To sync across devices, create a GitHub Gist named budget-data.json, copy the Gist ID from the URL, and generate a GitHub token with the gist scope. Enter them above and click Save.
